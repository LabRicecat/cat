# cat 0.0.6
Cat Approved TUI - a TUI framework, meow~

## Requirements
- C++ 20 (Language)
- [catcaretaker](https://github.com/LabRicecat/catcaretaker) (Up-To-Date Dependencies)
- cmake (Building)
- make (Building)
- ncurses (Display library)

## Installation
```
$ git clone https://github.com/LabRicecat/cat
$ cd cat
$ catcare sync
$ mkdir build && cd build
$ cmake ..
$ sudo make install
```
